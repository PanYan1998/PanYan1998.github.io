# README from Lixiang

## hexo常用命令

```
hexo new "name"       # 新建文章
hexo clean            # 清楚缓存文件
hexo g                # 生成静态网页
hexo d                # 部署
hexo s                # 本地预览（访问39.97.123.128:4000）
```
